About

The first date of data available from my data source (Johns Hopkins Univ CSSE) is January 22, 2020 and the most recent date that has complete data is two days ago -- the dataset is updated daily. Only the 50 countries with the highest number of total cases are shown in the worldwide country graph.

This visualization came from my own frustration around the poor management and visibility into tracking COVID-19 cases. I am working with a group of epidemiologists and technologists to create a more comprehensive technology solution to address COVID-19 through contact tracing via a voluntary, privacy-centric smartphone app. If you want to learn more, check out our site. If you want to get involved, email me (listed below).

This is not the most robust data visualization of COIVD-19 in the world. If you are looking for more data dashboards for COVID-19, the best I have found this dashboard from Johns Hopkins (the same source as the data used in my site here).
